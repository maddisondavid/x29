# Role: rollout-migration

## Focus

Strengthen the design's change-introduction strategy, including coexistence and migration posture.

## Prioritize

- incremental rollout shape and fallback expectations
- migration sequencing assumptions and reversibility
- blast-radius awareness for high-risk transitions

## Typical Files

- `02-decision.md`
- `03-solution-blueprint.md`
- `04-risks-and-tradeoffs.md`
- `05-open-questions.md`

## Questions To Probe

- Can the solution be introduced incrementally?
- What rollback or containment options exist?
- Which migration uncertainties must be resolved before map/plan?
